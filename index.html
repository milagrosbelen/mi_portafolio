Mis Proyectos

Proyecto "Turnos Belenza"

Turnos Belenza es una aplicación web diseñada para facilitar la reserva de turnos en un salón de uñas y belleza. El proyecto busca optimizar la experiencia del usuario, permitiendo elegir servicios, agregar extras, especificar preferencias y confirmar la cita de forma clara y rápida. El diseño es limpio, delicado y accesible desde cualquier dispositivo. Se enfoca en la experiencia móvil, priorizando la facilidad de navegación y la estética profesional de la marca. Este proyecto fue desarrollado de forma independiente con HTML, CSS y JavaScript puros. El formulario genera automáticamente un mensaje para confirmar el turno vía WhatsApp, permitiendo una gestión ágil para el negocio.

Tecnologias:Html, CSS, JavaScript

Proyecto "eBook: Crecé en Redes"

Este proyecto consiste en una landing page diseñada para la promoción y venta de un eBook digital titulado “Crecé en Redes con Éxito”, una guía práctica que enseña cómo destacarse en plataformas como Instagram, TikTok y más. El sitio presenta la información clave de forma estructurada y atractiva: beneficios del eBook, resumen del contenido, precio, llamado a la acción (botón de compra) y datos de contacto. Todo el diseño está pensado para convertir visitantes en compradores. Se priorizó una estética limpia y profesional utilizando una paleta en blanco y celeste claro, con un diseño responsive que se adapta perfectamente a cualquier dispositivo.

Tecnologias:Html, CSS
